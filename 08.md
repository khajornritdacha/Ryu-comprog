## Sorting

ถ้าหากอยาก sort [a, b, c] โดย a จากมากไปน้อย b จากน้อยไปมาก และ c จากมากไปน้อยให้สร้าง list ที่เป็น [-a, b, -c] แล้ว เช่น

```python
arr = [[10, 5, 7], [1, 2, 3], [5, 4, 3]]
# แปลงให้เป็น [-a, b, c]
arr = [[-10, 5, -7], [-1, 2, -3], [-5, 4, -3]]

arr.sort()
print(arr)

# output
# [[-10, 5, -7], [-5, 4, -3], [-1, 2, -3]]
```

## Sorted Vs list.Sort

### Sorted

- return new sorted list but not modified original list

```python
arr = [10, 5, 7]
print(sorted(arr))
print(arr)
# output
# [5, 7, 10]
# [10, 5, 7]
```

### list.Sort

- modified original list

```python
arr = [10, 5, 7]
arr.sort()
print(arr)
# output
# [5, 7, 10]
```

## Slicing

- slice list or list-like object
- syntax `list[start:stop:step]`

### Example

```python
num="0123456789"
print(num[0:5])
print(num[0:5:2])

# output
# 01234
# 024
```
